 Overview

This repository contains the complete replication package for the research paper:

> **"Narrative Disclosure Tone and Financial Reporting Quality: Evidence from AI-Based Textual Analysis Using NLP and Transformer Models"**

The study examines the association between narrative disclosure tone and financial reporting quality using advanced artificial intelligence (AI) and natural language processing (NLP) techniques. Unlike prior literature that relies on dictionary-based sentiment measures, this study employs transformer-based models (FinBERT) and ChatGPT-assisted classification to construct refined tone measures from annual report disclosures.
 Key Features

- FinBERT-based sentiment analysis** for context-sensitive tone measurement
- ChatGPT-assisted classification** for cross-validation and reliability enhancement
- Entropy-based instrument validation** for assessing instrument strength
- Comprehensive identification strategy** (IV-2SLS + System GMM)
- Fully reproducible Python pipeline** for end-to-end analysis
